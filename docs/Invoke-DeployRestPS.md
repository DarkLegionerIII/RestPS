# Invoke-DeployRestPS

## SYNOPSIS

## SYNTAX

```
Invoke-DeployRestPS [[-LocalDir] <String>] [<CommonParameters>]
```

## DESCRIPTION
This function will setup a local Endpoint directory structure.

## EXAMPLES

### EXAMPLE 1
```
Invoke-DeployRestPS -LocalDir c:\RestPS
```
